# 📖 TDD-Project <a name="about-project"></a>

> A demonstration of Test Driven Development, creating the tests for the methods of a Solver Class and then building said class.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- [Ruby](https://www.ruby-lang.org/en/)
- [Rubocop](https://docs.rubocop.org/rubocop/1.39/index.html)
- [Rspec](http://rspec.info/)

<!-- Features -->

### Key Features <a name="key-features"></a>

The Solver class can calculate factorials, reverse strings and solve the fizzbuzz problem.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: ``` Ruby ```

### Setup

Clone this repository to your desired folder.

### Install

Install this project with:

```

```
  rspec spec
